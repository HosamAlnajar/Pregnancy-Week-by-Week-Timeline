A few notes about the sections above:

== Installation ==

1. Activate the plugin through the 'Plugins' menu in WordPress
1. Add `[get_timeline_template]` in your post or page you want to display it in.
1. Place `<?php echo do_shortcode('[get_timeline_template]'); ?>` in your templates
